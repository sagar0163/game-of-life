# Conway's Game of Life

Classic cellular automaton in your terminal.

## Features

- 🎮 Interactive simulation
- 🎲 Random patterns
- ⏸️ Pause/Resume
- 🏗️ Multiple patterns

## Requirements

```bash
pip install keyboard
```

## Run

```bash
python life.py
```

## Controls

- Space: Pause/Resume
- R: Randomize
- C: Clear
- Q:

Sagar Jad Quit
